# CPSCRecallsBot
CPSC Recalls Bot

Written in Python 3

Lives at https://www.reddit.com/r/CPSCRecalls

*I use [Sentry](https://www.sentry.io) for logging errors*

## Configuration

The file `secret.txt` contains reddit access information. Syntax:

    Reddit Client ID
    Reddit Client Secret
    Reddit Username
    Reddit Password
